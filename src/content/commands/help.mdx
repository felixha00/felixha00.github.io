help — available commands

- `help` — Show available commands  
- `clear` — Clear the terminal output  
- `echo` \<text\> — Print text to the terminal
- `date` — Show the current date and time  
- `cd` \<page\> — Navigate to a different page
- works — Open the Works page  
- works experience — Go to Experience section  
- works software — Go to Software Projects  
- works hardware — Go to Hardware Projects  
- works design — Go to Graphic Design  
- works content — Go to Content Creation  
- `theme` \<dark | light\> — Switch to a different theme  
- `man` - Shows a "man"
